# ft_package

A very light Python utility package containing simple functions to work with
strings, numbers and lists.

## Features

- **Math Utilities**: Check even/odd, compute factorial, test for prime
- **String Utilities**: Reverse text, count vowels, capitalize words
- **List Utitilites**: Remove duplicates, flatten nested lists, chunk lists

## Installation

The package will be installed via pip using one of the following commands (both should work):

```bash
pip install ./dist/ft_package-0.0.1.tar.gz
```

or

```bash
pip install ./dist/ft_package-0.0.1-py3-none-any.whl
```

## Environment needs for installation

If you encounter an externally-managed-environment error, create a virtual environment first:

```bash
python3 -m venv venv
source venv/bin/activate
pip install ./dist/ft_package-0.0.1.tar.gz
```
